1. 反编译器会将所有变量定义在文件前部，首先收集所有变量，tree-sitter对应的变量定义语句的type为（declaration），变量的type为（identifier）
1.1 函数参数中变量定义为（parameter_declaration）
2. 收集函数调用（call_expression），判断输入函数，获取接收变量
3. 参数（第一个parameter_list）和接收变量均为输入变量
4. 查看输入变量的赋值情况，需要查看赋值语句（assignment_expression）
5. 比较语句（binary_expression），进去之后根据符号判断是否为比较

整体：
针对每条路径数据流分析
路径遍历方式：深度遍历

输入函数有：
scanf

遍历时变量状态会发生改变，需要提前保存状态，在其中一个分支返回时恢复状态
